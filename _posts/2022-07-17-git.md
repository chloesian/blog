```bash
git status
: 파일 상태를 알아보는 명령어

git add .
: 수정/변경된 파일이 있을 때 변경사항을 저장해주는 명령어

git commit -m"commit이름"
git push origin master
: status에 있던 파일들 사라지고 파일에 변경사항들이 적용됨
```
